Objectives
 
  • Provide a simple yet powerful JAVA API to solve single player puzzles such as "Sudoku", "Traveling Salesman Problem", "Rubik's cube", "Missionaries and cannibals" etc..
  • Provide educational value to people interested in learning AI.
  • Solve some cool puzzles.
 
Getting Started
 
  • Read the book "Artificial Intelligence: A Modern Approach" by Stuart J. Russell, Peter Norvig. Its an excellent read and provides a great introduction to AI. To understand this code, you just need to read the first 4 chapters!
  • Check out the source from svn and try our demos.
  • If you have any questions, contact me (Raghavendra Kotikalapudi) @ ragha_unique2000 [AT] yahoo [DOT] com.
